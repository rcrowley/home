Introducing Doubledown

This is not a sandwich.  I wrote a file-syncing utility called Doubledown designed for developers running code on a remote server (a DevStructure server, perhaps) but coding locally in their favorite editor or IDE.  Read all about it on the DevStructure blog.  If you already know this is what you want for Christmas: doubledown-0.0.1.tar.gz.
